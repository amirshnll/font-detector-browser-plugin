# Privacy Policy

**Font Detector** — last updated: July 1, 2025

## Summary

Font Detector does not collect, store, or transmit your personal data. Everything runs locally in your browser.

## What the extension does

When you click the extension icon, it scans the **currently active tab** to:

- List font families used on the page
- Find downloadable font files (woff, woff2, ttf, otf, eot)

Scan results are shown only in the popup. They are not saved or sent anywhere.

## Data we do not collect

Font Detector does **not**:

- Send data to external servers
- Use analytics or tracking
- Create user accounts
- Store browsing history
- Share information with third parties

## Permissions

| Permission   | Why it is needed                                      |
| ------------ | ----------------------------------------------------- |
| `activeTab`  | Access the page you are viewing when you open the popup |
| `scripting`  | Run the font scanner on that page                     |
| `downloads`  | Save font files you choose to download                |

These permissions are used only when you interact with the extension.

## Downloads and clipboard

- **Downloads** — when you click Download, the file is saved through your browser’s built-in download feature. The extension does not upload the file anywhere.
- **Copy** — when you copy a font name, it goes to your system clipboard only.

## Changes

If this policy changes, the updated version will be published in this repository.
